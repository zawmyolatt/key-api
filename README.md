## Requirements

- PHP 7.3 is installed. 
- For safety, PHP 7.3 or above , since that is the requirement of Laravel 8
- Make sure composer is installed and run `composer install` in project directory

## Running the tests

```
vendor/bin/phpunit

vendor/bin/phpunit --testdox
```
